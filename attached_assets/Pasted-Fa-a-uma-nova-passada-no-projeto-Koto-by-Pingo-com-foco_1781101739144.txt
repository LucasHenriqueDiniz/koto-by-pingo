Modo 1: Flashcards

- Botão "Ver resposta".
- Ao revelar, mostrar:

  - romaji
  - tradução PT-BR
  - categoria
- Botões:
  - Acertei
  - Errei
  - Próxima
- Registrar progresso por palavra.

Modo 2: Seleção de palavras
Este modo deve funcionar assim:
- Mostrar a palavra japonesa.
- Mostrar tradução como dica, se a configuração estiver ativada.
- Perguntar: "Qual é a leitura?"
- Mostrar opções de leitura em romaji.
- O usuário escolhe a leitura correta.

Exemplo:
Palavra: いえ
Dica: casa
Opções:
- ie
- iie
- e
- iya

Adicionar configuração local:
- showTranslationHint: boolean

Na UI:
- Toggle "Mostrar tradução como dica"
- Se desligado, mostrar só a palavra japonesa.

O modo deve registrar:
- wordId
- selectedOption
- correctOption
- result correct/wrong
- weakReason: "reading"

Modo 3: Combinar pares
Este modo deve funcionar com pares.

Mostrar duas colunas:
Coluna esquerda:
- palavras japonesas

Coluna direita:
- leitura + tradução

Exemplo:
Esquerda:
- いえ
- みず
- ねこ
- たべる

Direita:
- mizu — água
- ie — casa
- taberu — comer
- neko — gato

Usuário clica em um item da esquerda e depois em um item da direita.
Quando formar par:
- se correto, marcar como correto
- se errado, mostrar feedback leve
- ao finalizar todos, mostrar resumo

Adicionar configuração:
matchingPairMode:
- "reading"
- "meaning"
- "reading_and_meaning"

Padrão:
"reading_and_meaning"

Registrar progresso por palavra:
- correct/wrong
- weakReason:
  - "reading" quando o erro for leitura
  - "meaning" quando for tradução
  - "reading" como padrão para reading_and_meaning

Modo 4: Quiz de tradução
- Mostrar palavra japonesa.
